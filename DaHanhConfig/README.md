# Dạ Hành Studio - Config Repository

Kho lưu trữ cấu hình từ xa cho Extension Dạ Hành Flow.

### Các file cấu hình:
- `setting-config.json`: Toàn bộ XPaths và cài đặt automation.
- `news-popup-config.json`: Cấu hình thông báo và khuyến mãi.
- `vi.json`: Bản dịch tiếng Việt.
- `version.json`: Thông tin phiên bản cấu hình.
