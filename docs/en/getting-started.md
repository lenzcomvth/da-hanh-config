## ğŸ¬ Create Your First Video Batch

### Step 1: Open Google Flow

1. Navigate to **labs.google/fx/tools/flow**
2. Sign in with your Google account
3. You'll see the Flow video creation interface

### Step 2: Open SnapGen Flow Extension

1. Click the **extension icon** in your Chrome toolbar (puzzle piece icon)
2. Find **SnapGen Flow** and click to open the side panel
3. The extension will appear on the right side of your screen

> ğŸ’¡ **Tip:** If you don't see the extension, make sure you're on the Flow page first!

### Step 3: Prepare Your Prompts

Choose one of these methods:

**Option A: Type Manually**
- In the **Prompt List** textarea, type one prompt per line
- Example:
  ```
  A cat playing piano in a cozy room
  Sunset over mountains with flying birds
  City street at night with neon lights
  ```

**Option B: Import from File**
- Click **"Import from file (.txt)"**
- Select your `.txt` file with prompts (one per line)
- Prompts will auto-fill in the textarea

### Step 4: Configure Settings (Optional)

Basic settings you might want to adjust:

- **Videos per prompt**: How many variations per prompt (default: 1)
- **Aspect Ratio**: 16:9 (landscape), 9:16 (vertical), or 1:1 (square)
- **Start from**: Which prompt to start from (useful if continuing a batch)

### Step 5: Start Automation

1. Click the **green "Start" button**
2. Sit back and watch! The extension will:
   - Submit prompts one by one
   - Wait for videos to generate
   - Download completed videos automatically
3. You can **pause** or **stop** anytime

### Step 6: Check Your Videos

- Videos are saved to your Chrome **Downloads** folder
- File names include prompt number and timestamp
- Check the **Logs tab** for detailed progress
